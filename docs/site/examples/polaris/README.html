Polaris - A New Product Development Case Study

This example shows sample path analysis applied to a new product development process. The context for this analysis can be found in our post Little’s Law in a Complex Adaptive System

The specific scenario that we discussed in detail in that post can be found in the completed-stories-tukey directory.

In addition to those scenarios, we also have the following scenarios

  • The stability analysis for the defects sub-process
  • The stability analysis for all completed stories without outlier filtering
  • The divergence of aging work showing the stablity impact of unbounded open work.
  • The stabilization of complete epics
  • The divergence of aging epics
